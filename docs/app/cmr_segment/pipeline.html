CMRSegment processing pipeline

run(data_dir: Path)

Run DataPreprocessor -> Segmentor -> extract_simple_landmarks() -> SegmentationRefiner -> extract_simple_landmarks()-> MeshExtractor -> (Coregister) -> MotionTracker.

The pipeline can be run using only part of the modules, and this is set by the config.

Parameters:

data_dir – directory that contains subdirectories, where each one containing at least LVSA.nii.gz is a subject to be processed.

Returns:

None
